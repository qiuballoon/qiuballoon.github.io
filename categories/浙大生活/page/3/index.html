书卷多情似故人

“别说你爬过的山，只有早高峰。”看到这句广告，想到早高峰的公交。车内拥挤的乘客懒懒挪动，车下的乘客又难挤上去。司机时常难按暴躁地喊“往里走！上车的快点！”

在车上找到个立足之地后总忍不住想“瘦瘦瘦！瘦无止境！瘦瘦瘦！瘦成闪电！”总之这种时候会觉得减肥简直是造福人类。

这学期又结束了，其实不大愿意说这学期不开心好像比开心多了那么一点点，可是我真的过了这样的一学期啊，又骗不了自己…

一个学期没回家，最近老想回家。很想见见那些烂熟的人，然后拉着他们一股脑儿说很多很多话。可惜我不会喝酒，不然一杯酒就着一块猪头肉，往事好比酒肉穿肠过。

闲时还是习惯看看书打发时间，毕竟梦里不知身是客，而书卷多情似故人。

哎呀，说什么胡话啊，不过想家嘛。

然后我真的请了好几天假，下周回一趟家，回来再继续爬“早高峰”。

同行的人会同行

同行的人会同行

又到了打开朋友圈就能看到各种毕业照的季节。做晚 X 来跟我讲了离校的几个瞬间。有那么点似曾相似的感觉。

想起两年前大学毕业的时候，有个同学给我发了条短信“虽然这几年都没有你的消息，但始终无法忘记你，转眼就要毕业。”我两年多都没回他一条消息，终于在毕业前回了句“谢谢你还惦记。生活总难免遗憾，但愿我们拥有的都珍惜，得不到的都释怀。祝毕业快乐，一路坦途！”

我跟室友说，我年轻的时候有点“狠”，不能接受别人对我的态度踩在暧昧的灰色地带，就直接打个电话禁止对方以这样的方式对我。QQ 微信短信都不足以表达我立场的坚定，必须打电话说清楚，虽然心里总有忐忑，但一根筋地觉得要果断，不要消耗人家。表态结束后我就不回对方消息了。室友说，那后来你怎么不这样做了。

大脑的自由

如果学校建在铁轨旁，火车行驶的噪声会降低学生的学习效率是显而易见的。同样，大脑的噪声也会影响学习效率，这种情况更常见，却常常被忽视。

看了《稀缺》觉得贫穷近乎绝望，而不止是金钱的稀缺，忙碌者的时间稀缺或是其它形式的各种稀缺都一样会导致管窥心态、带宽负担、自我控制力和执行力下降等不利影响。

谁让大脑能处理的信息有限呢？一旦被“稀缺”俘获必然要牺牲对更深更远的信息的加工。

好吧，我只是觉得自己最近又陷入了阶段性疲惫，与忙碌和劳累都无关的无形疲惫。思考毕业论文如何设计、毕业后何去何从的问题就很容易陷入这样的状态。

也许这也是某种形式的稀缺吧，可以是时间，可以是自信心，可以是其它任何我实际拥有的少于我想要拥有的事物。所以我希望自己从头脑中的这些噪声中解放出来，还大脑以自由。

大脑的自由是效率和创造的保证。

一封信写给那些没那么开心的家伙

嘿，家伙，你选择打开这封信呀，可能因为你最近没那么开心，也可能你经常就没那么开心，当然也可能是出于好奇。

如果是前两者，那你很有勇气呀！怎么说呢，所有声音告诉你要开心生活，你怎么能说自己没那么开心呢？正如，所有声音都要求你做个成功的人，你怎么能说自己没那么成功呢；所有声音都说要融入集体，你怎么能说自己孤独呢？

要正视自己和主流声音不符是件需要勇气的事，大部分人早就习惯了想要安全的活着就要隐藏自己的一部分情感。

立夏

下了几天雨，今天总算是晴天。休息了一周没到实验室，今天决定回归学术。刚好大师兄在给大师姐传授结婚经验。还吃了个大师兄结婚的卤蛋。才知道送新婚夫妇鸡蛋寓意着“早生贵子”。

到咖啡屋还了上周借的《人类简史》，又借了本《稀缺》。晚饭后自己到草坪散散步，晚上又到操场跑了几圈。觉得自己的生活就是在学术、读书、吃饭、散步、跑步中循环…生活节奏很慢，很平静，却也怡然自得。

有些时候付出了会怕没有得到期待的结果，但也会跟自己说，无论如何，我努力过呀，每一次尝试都能换来问心无愧的踏实。

立夏了，但愿一切都是满怀希望的，像这个充满生机的季节，像树上慢慢浓郁的绿，像夏天欢快的虫儿鸣…

八张卡片：无聊只是自己的感受

有天晚上去跑步，不经意间看到清澈透明的圆月，想起以前看到的一句话—— “他说，月亮是湖，他的爱是舟”，如此美妙，如此灵动。

忽然明白什么眼前的苟且，远方和诗，哪有分得这么清楚。生活更像它们的杂糅，一边安顿眼前的苟且，一边安抚看似虚无的诗。

在觉得生活无聊的时候不妨提醒下自己，生活本身并没有无聊不无聊，无聊只是自己的感受。

这个时候你可以翻翻书、喝喝茶，去感受片刻宁静的美好； 也可以出去走走，看一株小草、一朵小花、一只小狗，去感受生命的脉搏。

最近把日常生活中拍的几张照片，写的几句话做成卡片留存，也收集在这里。
